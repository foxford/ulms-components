# Panel

Common panel element. Is used to split the application to the different sections.


#### with children

<Canvas>
  <Story name="with children" args={{}}>
    {
      <Panel debug>
        We gonna space?!
      </Panel>
    }
  </Story>
</Canvas>


#### with built-in children

<Canvas>
  <Story name="with build-in children" args={{}}>
    {
      <Panel title='We gonna space?!' debug />
    }
  </Story>
</Canvas>


#### with custom children

  <Story name="with custom children" args={{}}>
